ESG Insight API

ESG Insight API를 통해 ESG 진단, 탄소 배출 계산, 보고서 생성 기능을 귀사의 시스템에 직접 연동할 수 있습니다.

Base URL

https://api.esginsight.co.kr/v1

인증

모든 API 요청에는 인증이 필요합니다. API 키를 HTTP 헤더에 포함하여 요청하세요.

Header 
Authorization: Bearer YOUR_API_KEY

보안 주의

API 키는 절대 클라이언트 사이드 코드에 노출되지 않도록 주의하세요.

ESG 진단

GET /diagnosis

현재 ESG 진단 결과를 조회합니다.

Response

{
  "status": "success",
  "data": {
    "overall_score": 72,
    "environment": {
      "score": 68,
      "items_completed": 17,
      "items_total": 17
    },
    "social": {
      "score": 75,
      "items_completed": 22,
      "items_total": 22
    },
    "governance": {
      "score": 73,
      "items_completed": 22,
      "items_total": 22
    },
    "updated_at": "2024-01-15T09:30:00Z"
  }
}
POST /diagnosis/answers

진단 항목에 대한 답변을 제출합니다.

Request Body

{
  "item_id": "E-1-1",
  "answer": "implemented",
  "evidence": "환경경영시스템 ISO14001 인증서",
  "note": "2023년 갱신 완료"
}

탄소 배출

POST /carbon/calculate

에너지 사용량을 입력하여 탄소 배출량을 계산합니다.

Request Body

{
  "period": "2024-01",
  "electricity_kwh": 15000,
  "gas_m3": 500,
  "diesel_l": 200,
  "gasoline_l": 150
}

Response

{
  "status": "success",
  "data": {
    "period": "2024-01",
    "total_emissions_tco2e": 8.54,
    "scope1": 2.31,
    "scope2": 6.23,
    "breakdown": {
      "electricity": 6.23,
      "gas": 1.12,
      "diesel": 0.53,
      "gasoline": 0.35
    }
  }
}

보고서

POST /reports/generate

ESG 보고서를 생성합니다.

Request Body

{
  "type": "k-esg",
  "format": "pdf",
  "include_carbon": true,
  "include_benchmarks": true,
  "language": "ko"
}

Available Report Types

k-esg - K-ESG 보고서
gri - GRI 보고서
tcfd - TCFD 보고서
carbon - 탄소 배출 보고서

웹훅

웹훅을 설정하면 보고서 생성 완료 등 이벤트 발생 시 지정된 URL로 알림을 받을 수 있습니다.

지원 이벤트

  • report.completed 보고서 생성 완료
  • diagnosis.updated 진단 결과 업데이트
  • carbon.calculated 탄소 배출량 계산 완료
