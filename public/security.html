데이터 암호화

모든 데이터는 전송 중(TLS 1.3)과 저장 시(AES-256) 암호화됩니다.

  • TLS 1.3 프로토콜로 전송 데이터 암호화
  • AES-256 알고리즘으로 저장 데이터 암호화
  • 비밀번호 bcrypt 해시 처리

접근 제어

역할 기반 접근 제어(RBAC)와 다중 인증(MFA)을 통해 무단 접근을 방지합니다.

  • 2단계 인증(2FA) 지원
  • SSO(Single Sign-On) 연동
  • IP 화이트리스트 설정 가능
  • 세션 타임아웃 자동 로그아웃

인프라 보안

AWS 서울 리전에서 운영되며, 글로벌 수준의 물리적·네트워크 보안을 갖추고 있습니다.

  • AWS 서울 리전 (ap-northeast-2)
  • VPC 격리 및 프라이빗 서브넷
  • WAF (웹 애플리케이션 방화벽)
  • DDoS 방어 (AWS Shield)
